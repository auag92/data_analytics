Linear and Non-linear Regression
Data Analytics Course, Aug 2015

Ramesh Hariharan

1

The Regression Problem

  • Given an (unknown) function g(x) evaluated at n points x1…xn

  • The goal is recover g

  • Of course, g is not specified uniquely by these n evaluations

  • So assume g is of a simple, given form, say ax+b or ax2+bx+c, where a,b,c are unknown parameters

  • Find the a,b,c that match the evaluations most closely

Ramesh Hariharan

2

The Regression Problem More Precisely

  • Given n pairs xi,yi

  • xi∈Rd, yi∈R

  • Given a model fΘ(x), x∈Rd, Θ represents the unknown parameters

  • Find Θ that minimizes h(Θ)=∑i|fΘ(xi)−yi|2

Ramesh Hariharan

3

Linear Regression

  • Assume fΘ(x)=xTΘ

  • Θ,x∈Rd

  • Find Θ that minimizes h(Θ)=∑i|xTiΘ−yi|2

  • Set ∂h∂Θ to 0

  • Check that this is indeed a minimum, not a maximum, not a saddle etc

Ramesh Hariharan

4

Determining Parameters in Linear Regression

  • Rewriting more cleanly..

  • h(Θ)=(XΘ−y)T(XΘ−y), where X is n×d, the ith row of X is xi

  • dhdΘ=2(XTXΘ−XTy), where dhdΘ is a vector of partial derivatives w.r.t each of the entries in the vector Θ

  • Setting these to 0, we get a system of d equalities XTXΘ=XTy. Solution: (XTX)−1XTy

Ramesh Hariharan

5

Global Minimality

  • Invertibility? One solution if invertible, many otherwise. Why?

  • Each solution of the form Θ=(XTX)−1XTY is a global minimum. Why?

  • By Taylor's expansion for multivariate functions, for any vector Δ=(Δ1,Δ2,…,Δd+1)∈Rd+1,

    h(Θ+Δ)=h(Θ)+ΔTh′(Θ)+12ΔTh′′(Θ)Δ

  • Higher order terms are 0 for linear regression

Ramesh Hariharan

6

Global Minimality

  • h′(Θ)=⎛⎝⎜⎜⎜⎜⎜⎜∂h∂Θ1∂h∂Θ2⋮∂h∂Θd⎞⎠⎟⎟⎟⎟⎟⎟, h′′(Θ)=⎛⎝⎜⎜⎜⎜⎜⎜⎜∂2h∂Θ1∂Θ1∂2h∂Θ2∂Θ1⋱∂2h∂Θd∂Θ1∂2h∂Θ1∂Θ2∂2h∂Θ2∂Θ2∂2h∂Θd∂Θ2⋯⋯⋯∂2h∂Θ1∂Θd∂2h∂Θ2∂Θd∂2h∂Θd∂Θd⎞⎠⎟⎟⎟⎟⎟⎟⎟

  • We chose Θ so h′(Θ)=0.

  • Claim: ΔTh′′(Θ)Δ≥0, which suffices for global minimality

Ramesh Hariharan

7

Global Minimality and Uniqueness

  • h′′(Θ), also called the Hessian, is exactly XTX!

  • ΔTXTXΔ=|XΔ|2≥0

  • If X (which is n×d, n≥d) has full rank d then XΔ is always non-zero, so |XΔ|2>0, i.e., there is a unique global minimum

  • Otherwise, any Δ in the nullspace of X satisfies XΔ=0 and h(Θ+Δ)=h(Θ), so moving around in the nullspace yields all the global minima

Ramesh Hariharan

8

Invertibility of XTX

  • rank(X)=rank(XTX). Why? Because the nullspaces are the same.

  • If X has rank d then XTX is invertible and there is an unique global minimum

  • If X has smaller rank, then XTX is not invertible, then how do we find solutions to XTXΘ=XTy?

  • Note XTX takes an arbitrary vector Θ and transforms it into the column space of XT.

Ramesh Hariharan

9

The Pseudo-Inverse of XTX

  • Since XTX is a real, symmetric, positive semi-definite d×d matrix, all its eigenvalues are real and non-negative

  • The corresponding eigenvectors are either in the column space of XT or orthogonal to it, and form an orthonormal basis

  • Eigenvector v has eigenvalue 0 if it is orthogonal to all the columns of XT and eigenvalue non-0 if it is in the column space of XT

  • XTX=∑ieivivTi where ei are the non-0 eigenvalues and vi denotes the corresponding eigenvectors

Ramesh Hariharan

10

The Pseudo-Inverse of XTX Contd.

  • Consider (∑i1eivivTi)XTy as a solution for Θ

  • Claim: (XTX)(∑i1eivivTi)XTy=XTy

  • Proof:

    • (∑ieivivTi)(∑i1eivivTi)XTy=(∑ivivTi)XTy
    • (∑ivivTi)x=x for any vector x in the column space of XT
    • Since XTy is in the column space of XT, (∑ivivTi)XTy=XTy, as needed

Ramesh Hariharan

11

Summarizing the Non-Invertible Case

  • Since the column space of XT has dimension at least 1, at least one eigenvalue of XTX is non-zero, hence there is at least one solution satisfying h′(Θ)=0, namely Θ=(∑i1eivivTi)XTy

  • This is a global minimum, though not unique, any h(Θ+Δ) where Δ is in the nullspace of X (equvalently, orthogonal to column space of XT) is also a global minimum

  • There is a unique global minimum in the column space of XT

Ramesh Hariharan

12

Running Time

  • Compute XTX

  • Compute eigenvalues and eigenvectors

  • Compute (∑i1eivivTi)XTy

  • Naively, O(d2n+d3)

  • There are ways to compute XTX faster, approximately, via random sampling

Ramesh Hariharan

13

The Non-Linear Case

  • Say fΘ(x)=(xTΘ)2 but not fΘ(x)=(x2)TΘ , where x2 denotes the vector x with entries squared; assume well-behaved (continuously differentiable)

  • h(Θ+Δ)=h(Θ)+ΔTh′(Θ)+ΔTh′′(Θ)Δ+O(Δ3) for small enough Δ

  • h′(Θ)=0 can no longer be solved using linear equations. Needs a general root-finding procedure

  • h′′(Θ) need not be of the form ATA, which means ΔTh′′(Θ)Δ could be negative, i.e., a root need not be local minimum, let alone a global one

Ramesh Hariharan

14

The Gauss-Newton Approach

  • Start with a particular Θ0

  • Linearize fΘ(x) in the neighborhood of this Θ0: fΘ0+Δ(x)∼fΘ0(x)+[f′Θ0(x)]TΔ

  • Minimize hh(Δ)=∑i|fΘ0(xi)+[f′Θ0(xi)]TΔ−yi|2

  • Use linear regression to find Δ=(ZTZ)−1ZT(y−fΘ0(x)) that minimizes hh(Δ); iterate with Θ0=Θ0+Δ

  • Where the ith row of Z is the vector [f′Θ0(xi)]T

Ramesh Hariharan

15

Convergence of the Gauss-Newton Approach

  • If fΘ0+Δ(x)∼fΘ0(x)+[f′Θ0(x)]TΔ is close to an equality then Θ0+Δ provides a strictly better fit than Θ0 unless Θ0 is already a local minimum

  • So if we start very close to a local minimum (so the linear approximation holds in a very close neighborhood), this will converge

  • But no guarantee if you start further away.

Ramesh Hariharan

16

Modified Gauss-Newton with Smaller Step Sizes

  • Δ=(ZTZ)−1ZT(y−fΘ0(x)), which equals −12(ZTZ)−1h′(Θ0) (why?)

  • h(Θ)=(fΘ(x)−y)T(fΘ(x)−y) and h′(Θ)=2ZT(fΘ(x)−y)

  • Θ0+αΔ=Θ0−α2(ZTZ)−1h′(Θ0)

  • For small enough α>0, h(Θ0+αΔ)∼h(Θ0)−α2h′(Θ0)T(ZTZ)−1h′(Θ0)<h(Θ0), at least if Z is full rank and Θ0 is not already a local minimum

Ramesh Hariharan

17

The Modified Gauss-Newton Algorithm for the Non-Linear Case

  • Start with some Θ0 and then move to Θ0−α(ZTZ)−1h′(Θ0)=Θ0−α(ZTZ)−1ZT(fΘ0(x)−y)

  • Here the ith row of Z is the vector [f′Θ0(xi)]T

  • α can be determined by single-parameter minimization

    • e.g., bisection: start with an interval, evaluate derivative in the middle, if negative, go right, otherwise go left

  • Stop when there isn't much progress

Ramesh Hariharan

18

Some More Modifications: Levenberg

  • Instead of Θ0−α(ZTZ)−1h′(Θ0) one could simply follow steepest gradient to Θ0−αIh′(Θ0)

  • The latter always leads to improvement but could be slow, the former is quicker if fΘ0(x) is linear in the neighbourhood of Θ0

  • Levenberg: Combine the two approaches, i.e., use Θ0−α(ZTZ+λI)−1h′(Θ0)

  • Adjust λ in each iteration, if there is quick descent then keep it low, otherwise increase it so you are closer to steepest descent

Ramesh Hariharan

19

Some More Modifications: Marquardt

  • Marquardt: Instead of Θ0−α(ZTZ+λI)−1h′(Θ0), use Θ0−α(ZTZ+λdiag(ZTZ))−1h′(Θ0)

  • diag(ZTZ)=⎛⎝⎜⎜⎜⎜⎜⎜⎜⎜⎜∑i(∂fΘ0(xi)∂Θ1)2(∑i∂fΘ0(xi)∂Θ2)2⋱(∑i∂fΘ0(xi)∂Θd)2⎞⎠⎟⎟⎟⎟⎟⎟⎟⎟⎟

  • When λ is large so one is in the steepest descent regime, move faster in directions where the partial derivative is smaller, in a bid to accelerate

Ramesh Hariharan
